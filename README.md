## ğŸš€ Instalasi & Cara Menggunakan

Ikuti langkah-langkah berikut untuk menjalankan project ini di local environment.

### 1. Clone Repository

```bash
git clone https://github.com/Adityammar65/FindSrv.git
cd FindSrv
```

Jika menggunakan XAMPP silahkan clone di htdocs.

### 2. Install Dependency

Pastikan **Composer** sudah terpasang, lalu jalankan:

```bash
composer install
```

### 3. Konfigurasi Environment

Salin file environment:

```bash
cp .env.example .env
```

Atur konfigurasi database dan setting lain di file `.env`.

### 4. Jalankan Server

```bash
php spark serve
```

Akses aplikasi melalui browser:

```
http://localhost:8080
```

Atau jika menggunakan XAMPP:

```
http://localhost/FindSrv/Public/
```
